LIBRARY MEMORY PERSONA LAB

图书馆“大学四年”用户画像与页面分流样例

这份看板用于判断不同学生的主等级、额外标签和锐评尺度。真实上线时，所有用户展示同一套页面；主等级按总次数、总天数、总小时数必选，早八、关灯、座位纯爱、考研等作为额外标签叠加。

当前先做样例，不绑定真实账号。确认后再接入登录、同步、主等级判定、额外标签判定和生成链路。

PRIMARY LEVELS

必要主等级分档

每个用户必须先落到一个主等级。低频阈值提高到少于 20 次；后续再根据时段、空间、期末、关键去向追加额外标签。

LEVEL 01 图书馆云股东

总预约 < 20，或活跃天 < 10，或总时长 < 60h。

LEVEL 02 偶尔闪现型选手

总预约 < 60，或活跃天 < 30，或总时长 < 180h。

LEVEL 03 稳定在线型同学

总预约 < 150，或活跃天 < 90，或总时长 < 700h。

LEVEL 04 深度常驻型选手

总预约 < 350，或活跃天 < 180，或总时长 < 1800h。

LEVEL 05 高强度驻馆人

总预约 < 500，或活跃天 < 260，或总时长 < 3000h。

LEVEL 06 人形自走卷王

总预约、活跃天、总时长全部进入最高档。
